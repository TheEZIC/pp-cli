#Usage

```
Usage:  [command] [options]

Options:
  -?|-h|--help              Show help information.
  -m|--mode <MODE>          Set osu! gamemode
                            Default value is: 0.
  -s|--score <SCORE>        Set score
                            Default value is: 1000000.
  -a|--accuracy <ACCURACY>  Set accuracy
                            Default value is: 1.
  -c|--combo <COMBO>        Set combo
                            Default value is: -1.
  -miss|--miss <MISS>       Set count of misses
                            Default value is: 0.
  -mods|--mods <MODS>       Set comma divided mods acronyms

Commands:
  file                      Get beatmap from .osu file
  id                        Get beatmap from osu API

Run ' [command] -?|-h|--help' for more information about a command.
```
